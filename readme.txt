How To Run
- "main.py" should locate outside of gsdmm repository which should be cloned from https://github.com/rwalk/gsdmm
- install requirements preferably in a virtual environment.
- Install NLTK stopwords data. Check out https://www.nltk.org/data.html to see how.
- from the code, change data file address & name accordingly.
- you may need to adjust parsing of your file in create_docs() method.
- This code is tested with Python 3.8.9
